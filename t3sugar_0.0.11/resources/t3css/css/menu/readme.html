Typo3 CSS Framework - Navigation
(de) Die Navigations-CSS-Klassen sollen das erstellen von Navigations-Elementen erleichtern. Sie bieten ein Grundgerüst für das richtige Skalieren und Fließen! Das jeweilige Navigationselement bekommt Farbe und Aussehen durch den ID-Selektor! Die Klasse ist ein Hilfsmittel für wiederauftretene Eigenschaften! Definiert eine Navigationslinie aus UL LI Tags. Alle Layout spezifischen Eigenschaften befinden sich im Ordner Obsolet! Dieser kann in der jeweiligen Include Datei entfernt werden!
